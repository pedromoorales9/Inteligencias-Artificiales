El Arte del Super Prompt

Domina el arte de crear prompts perfectos para obtener resultados excepcionales con IAs

Los prompts son la clave para obtener los mejores resultados de las IAs. Un buen prompt puede marcar la diferencia entre una respuesta mediocre y una excepcional.

Esta guía te enseñará cómo estructurar tus prompts de manera efectiva para cada tipo de IA.

Estructura del Super Prompt

Un Super Prompt es una estructura cuidadosamente diseñada que maximiza la efectividad de tus interacciones con IAs. Sigue esta estructura para obtener los mejores resultados:

Componentes Esenciales

[CONTEXTO] - Propósito: [Objetivo principal] - Audiencia: [Público objetivo] - Formato: [Formato deseado] - Tono: [Estilo de comunicación] [INSTRUCCIONES] 1. [Acción específica 1] 2. [Acción específica 2] 3. [Acción específica 3] 4. [Acción específica 4] [RESTRICCIONES] - [Limitación 1] - [Limitación 2] - [Limitación 3] - [Consideraciones especiales] [EJEMPLOS] Ejemplo positivo: [Lo que quieres] Ejemplo negativo: [Lo que no quieres] Referencia: [Ejemplo similar] [RESULTADO ESPERADO] - Formato: [Especificaciones detalladas] - Longitud: [Extensión deseada] - Tono: [Estilo específico] - Elementos clave: [Puntos importantes]

Consejos Avanzados

Especificidad

Sé lo más específico posible en tus instrucciones. Define claramente el formato, estilo y resultados esperados. Evita ambigüedades y términos vagos.

Contextualización

Proporciona suficiente contexto y antecedentes. La IA necesita entender el marco completo para generar respuestas relevantes.

Iteración

No temas refinar y ajustar tus prompts. La iteración es clave para encontrar la formulación perfecta para cada tarea.

Ejemplos por Tipo de Tarea

Redacción de Contenido

[CONTEXTO] - Propósito: Artículo técnico sobre IA - Audiencia: Desarrolladores junior - Formato: Blog post - Tono: Educativo y accesible [INSTRUCCIONES] 1. Explicar conceptos básicos 2. Incluir ejemplos prácticos 3. Añadir código de muestra 4. Proporcionar recursos adicionales [RESTRICCIONES] - Evitar jerga técnica compleja - Máximo 1500 palabras - Incluir imágenes descriptivas - Sin repeticiones [RESULTADO ESPERADO] - Formato: HTML con Markdown - Secciones claramente definidas - Enlaces a recursos - Conclusión accionable

Análisis de Datos

[CONTEXTO] - Propósito: Análisis de ventas - Datos: CSV trimestral - Formato: Informe ejecutivo - Audiencia: Directivos [INSTRUCCIONES] 1. Identificar tendencias clave 2. Calcular métricas principales 3. Generar visualizaciones 4. Proponer recomendaciones [RESTRICCIONES] - Datos confidenciales - Enfoque en ROI - Máximo 5 gráficas - Formato empresarial [RESULTADO ESPERADO] - Dashboard interactivo - KPIs destacados - Insights accionables - Proyecciones futuras
